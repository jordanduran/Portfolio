PROJECTS

Odd But True is a quiz application designed for any age developed using HTML, CSS, Javascript, and JQuery. It contains 10 random questions on various topics with multiple choice answers. The user answers as many as they can correctly and is presented with their final score and an option to play again at the end of the quiz.

Repo | Live

Grubclub is an application utilizing the Zomato API to display important restaurant information such as reviews, menu, photos, address, and more. The user inputs a type of food along with a location, a list of restaurants are then displayed with the important details. Created using HTML, CSS, Javascript, and JQuery.

Repo | Live

DevSpace is a social application created for developers anywhere in the world to be able to create their own profile along with their portfolio, and create posts. App was developed using HTML5, CSS3, React, Express, PostgreSQL, Heroku, and ZEIT for deployment.

Repo | Live

Shopzilla is a full-stack application where users are able to create an account to showcase their business. Sellers can provide their shop details and upload products to their page. Buyer accounts can explore the available shops and save items to their wishlist. This application was built with HTML5, CSS3, JavaScript, React, Nodejs, Express, and PostgreSQL for the database.
